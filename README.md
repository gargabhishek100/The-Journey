## 🎓 Key Projects

Full-stack AI-powered document processing platform
- **Frontend**: React with modern UI
- **Backend**: Node.js/Express with AI model integration
- **Features**: Intelligent extraction, model fallback rotation, cold-start optimization
- **Status**: Production-ready

### **Habit Forge** 🔥
Personal habit tracking and formation application
- **Stack**: TypeScript, Firebase Firestore
- **Features**: Real-time synchronization, security rules, comprehensive testing
- **Status**: Early stage with strong foundation

### **Portfolio** 🎨
Professional portfolio website and project showcase
- **Stack**: TypeScript/React, Vercel deployment
- **Features**: Responsive design, continuous deployment, performance optimization
- **Status**: Active with 30+ commits

### **Database Project (KUWSMP)** 🗂️
Document indexing and processing system
- **Stack**: Python/Streamlit, Firebase

- **Status**: Private repository with active development
